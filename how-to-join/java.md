# JAVA版の場合

## 参加情報 <a href="#join-address" id="join-address"></a>

`play.roomserver.blog`

## 参加方法

1. Minecraftを起動しましょう

<figure><img src="../.gitbook/assets/how-to-join-java-1.png" alt=""><figcaption></figcaption></figure>

2. マルチプレイを押すと、下記画像のようになります。

<figure><img src="../.gitbook/assets/how-to-join-java-2.png" alt=""><figcaption></figcaption></figure>

3. サーバーを追加/ダイレクト接続を押すと、下記画像のようになります。  
`play.roomserver.jp` をサーバーアドレスの欄に書きましょう！

<figure><img src="../.gitbook/assets/how-to-join-java-3.png" alt=""><figcaption></figcaption></figure>

4. **保存**を押し、接続ボタンを押すと接続することができます。  
**サーバーに接続** を押すと、接続できます！

サーバーに入って楽しみましょう！
