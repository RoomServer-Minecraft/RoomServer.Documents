# 📜 RoomServer プライバシーポリシー

最終更新日: 2026年2月23日

RoomServer（以下「本サーバー」といいます）は、ユーザーのプライバシーを尊重し、本サーバーおよび関連サービス（以下「本サービス」といいます）の提供にあたって取得する個人情報の保護に努めます。

### 第1条（取得する情報の項目）

**本サーバーは、本サービスの提供・維持・管理のために、以下の情報を取得する場合があります。**

1. アカウント情報
    * Minecraftユーザー名および固有識別番号（UUID）
    * Discordユーザー名およびユーザーID

2. 接続・利用ログ
    * 本サーバーへの接続日時および切断日時
    * 接続元IPアドレスおよびホスト名
    * ゲーム内でのチャットログ、コマンド実行履歴、行動ログ（建築・破壊・アイテム移動等）
    * Discordサーバー内での発言および行動履歴

3. 寄付に関する情報
    * 寄付の決済完了時に提供される決済ID（※クレジットカード番号等の決済詳細は、決済代行サービス側で管理され、本サーバーは直接保持しません）

### 第2条（情報の利用目的）

**取得した情報は、以下の目的のために利用します。**

1. サービスの運営および管理
    * ユーザーの識別、アカウントの認証、特典（VIPロール等）の付与。

2. セキュリティおよび荒らし対策
    * 利用規約違反行為（荒らし、チート等）の調査、特定および処罰。
    * 不正アクセスやサーバー攻撃の防止。

3. ユーザーサポート
    * トラブル解決のためのログ調査、問い合わせへの回答。

4. サービスの改善
    * サーバーの負荷状況の分析、システムの最適化。

### 第3条（第三者への提供）

**本サーバーは、以下の場合を除き、取得した情報を第三者に提供することはありません。**

1. ユーザー本人の同意がある場合。
2. 法令に基づく開示請求（裁判所、警察等からの公的な要請）があった場合。
3. サーバー内のセキュリティ維持のため、BANリスト等の共有プラットフォーム（他サーバーとの連携等）に、違反者のMinecraft IDや理由を登録する場合。
4. 人命、身体、または財産の保護のために緊急の必要がある場合。

### 第4条（情報の保護・管理）

1. 本サーバーは、取得した情報の漏洩、紛失、不正アクセスを防止するため、適切なセキュリティ対策を講じます。
2. 取得した情報にアクセスできるのは、OWNERおよび管理権限を付与された特定のSTAFFに限定されます。

### 第5条（外部サービスの使用）

**本サービスでは、以下の外部サービスを利用する場合があります。各サービスのプライバシーポリシーについては、それぞれの公式サイトをご確認ください。**

* Microsoft / Mojang: Minecraftサービスの基盤。
* Discord: コミュニティ管理。
* 決済代行サービス（PayPal, Stripe等）: 寄付の決済処理。

### 第6条（情報の開示・削除）

**ユーザーは、自身に関する情報の開示や削除を求めることができます。ただし、以下の場合は応じられないことがあります。**

1. 利用規約違反によるBAN処罰の記録（再入室防止のために保持が必要なため）。
2. 本サーバーの適正な業務運営に著しい支障を及ぼす恐れがある場合。
3. 法令に違反することとなる場合。

### 第7条（プライバシーポリシーの改定）

本サーバーは、法令の変更やサービス内容の変化に合わせて、本ポリシーを随時改定できるものとします。  
変更後のポリシーは、本サーバー内または公式サイト上に掲載された時点で効力を生じるものとします。
