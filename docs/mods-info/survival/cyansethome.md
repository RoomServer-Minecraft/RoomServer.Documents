# CyanSetHome

## CyanSetHomeとは？

「CyanSetHome」とは、設定したホームの場所に、いつでもテレポートができるMODです。  
信頼できる人に、ホームを共有することもできます。

## 補足

設定できるホームの上限は、今のところ**1人2個**までとなっております。  
知らない人にホームを共有しないよう、ご注意ください。
