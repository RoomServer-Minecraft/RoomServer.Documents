# Gsit

## Gsitとは？

「Gsit」とは、座ったり寝たりなど、プレイヤーの見た目のバリエーションを増やせるMODです。

## 使い方

* 階段やハーフブロックに右クリックをすると、座ることができます。
* プレイヤーに素手で右クリックをすると、そのプレイヤーの上に乗ることができます。
