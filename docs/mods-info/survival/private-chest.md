# Private Chests

## Private Chestsとは？

「Private Chests」とは、看板を設置するだけで、チェストや樽を他のプレイヤーから保護できるMODです。  
コマンドや複雑な設定は不要で、直感的に自分だけの収納スペースを作ることができます。

{% hint style="success" %}
**Private Chestsの主な機能は次の通りです**

* **看板による簡単な保護**: チェストに `[private]` と書いた看板を貼るだけでロックできます。
* **共有機能**: 看板にプレイヤー名を書くことで、特定のメンバーとチェストを共有できます。
* **強力なセキュリティ**: 不正な開封、ホッパーによる引き抜き、TNTやクリーパーの爆発から保護されます。
* **マルチプラットフォーム対応**: Bedrock Edition（統合版）のプレイヤーも、Geyser等を通じて利用可能です。
* **バニラ互換**: サーバー側のみのMODなので、プレイヤー側への導入は不要です。
{% endhint %}

## 使い方

### チェストを保護する

1.  保護したいチェストや樽に**看板**を設置します。
2.  看板の**1行目**に `[private]` と入力します。
3.  看板を閉じると、そのチェストはあなた専用になります。

### 他のプレイヤーと共有する

信頼できるプレイヤーを追加して、一緒にチェストを使えるようにすることができます。

1.  設置した看板の**2行目以降**に、許可したいプレイヤーの名前を入力します。
2.  1行に複数の名前を書く場合は、カンマ（`,`）で区切ります。
    *   例: `PlayerA, PlayerB`
3.  看板の**裏面**にも名前を書くことができ、合計で最大7人まで追加可能です。


## 補足

*   **二連チェスト対応**: 二連チェストの片方に看板を貼るだけで、全体が保護されます。
*   **荒らし対策**: 保護されたチェストの隣に勝手にチェストを置いて二連チェストにし、中身を抜き取るような不正行為も防止されています。
*   **爆発・火災保護**: 保護されたチェストそのものは、爆発や延焼の影響を受けません。
