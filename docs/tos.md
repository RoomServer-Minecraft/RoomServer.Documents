# 📜 RoomServer 利用規約

最終更新日: 2026年2月23日

### 第1条（総則）

* 本規約は、RoomServer（以下「本サーバー」といいます）が提供するMinecraftサーバーおよび関連するコミュニティ（Discordサーバー等を含む）を利用するすべてのユーザー（以下「ユーザー」といいます）に適用されます。
* ユーザーは、本サーバーに接続または参加した時点で、本規約および別途定めるプライバシーポリシーに同意したものとみなします。

### 第2条（定義）

* 運営: RoomServerの運営権限を持つOWNER（総括管理者）およびSTAFF（運営補助者）を指します。
* アカウント: ユーザーが本サーバーを利用するために使用するMinecraft ID（Java版/統合版）およびDiscordアカウントを指します。

### 第3条（禁止事項）

**ユーザーは、本サーバーの利用にあたり、以下の行為を行ってはなりません。**

* **不正ツール・チート行為**
    * X-Ray、マクロ、およびゲームバランスを崩壊させる不正クライアントの使用（※軽量化MOD等の通常プレイ補助は除く）。
    * ゲームのバグ（グリッチ）を意図的に利用する行為。

* **迷惑行為・荒らし**
	* 他者の建築物への破壊、着火、TNTの使用、窃盗行為。**ただし、特定のゲームモードで個別に定めるルールがある場合は、そのルールが優先されます。**
	* サーバーに過度な負荷をかける装置の作成や行為（Lag Machine等）。
	* 運営が指定した立入禁止区域（ネザー岩盤上など）への侵入。
	* 特定のサーバー負荷をかけるMOB（ウィザー等）の無許可召喚。

* **コミュニケーションに関する禁止事項**
	* 暴言、差別的発言、わいせつな表現、他者を不快にさせる行為。
	* 個人情報の開示（自身・他者問わず）。
	* 虚偽の情報の流布。
	* Discordにおける、フレンド以外のユーザーへの無断DM送信（勧誘・詐欺防止のため）。

* **不適切な集団形成**
	* 特定の個人やグループを攻撃、または疎外することを目的とした排他的な集団行動。
	* サーバーの秩序を著しく乱す派閥争い、および他ユーザーの参加を著しく拒む行為。

* **システム・運営に関する禁止事項**
	* 運営へのなりすまし、または代理問い合わせ行為。
	* 複数アカウント（サブアカウント）の無申告での利用。
	* VPNを使用した接続（※やむを得ない事情がある場合は事前申請が必要）。
	* スパム行為、詐欺行為、およびそれらの計画・幇助。
	* 指定チャンネル外での宣伝行為。

{% hint style="caution" %}
■ 爆発物の使用に関する特則
	•	オーバーワールド（現世） におけるTNT等の爆発物の使用は許可します。
ただし、他者の建築物の破壊、荒らし行為、サーバーへ過度な負荷を与える使用は禁止とします。
	•	ネザーおよびエンド におけるTNT等の爆発物の使用は、事前に所定の方法で申請し、運営の許可を得た場合に限り可能とします。
{% endhint %}
{% hint style="caution" %}
■ エンドラ討伐時の特例
	•	サーバーリリース後1週間後に実施されるエンドラ討伐イベントに限り、
運営が定める数時間の間、エンド内での爆発物使用を申請不要で許可します。
	•	特例時間終了後は、通常通り申請制に戻ります。
{% endhint %}
{% hint style="warning" %}
■ エンドの利用制限
	•	エンドは、サーバーリリース後1週間が経過するまで利用できません。
	•	上記期間中はエンドポータルは動作しません。
{% endhint %}
### 第4条（アカウント管理）

* **サブアカウントの利用**  
    サブアカウントの利用は原則禁止ですが、所定の申請フォームより事前報告を行い、運営の許可を得た場合に限り利用可能です。
* **アカウントの譲渡・共有**  
    アカウントを第三者に譲渡、貸与、または共有する行為は禁止します。

### 第5条（プレイヤー間のトラブル）

* **不介入の原則**  
    運営は、原則としてユーザー間の主観的なトラブル（感情的な対立、合意の上でのアイテム貸借による未返却、口論等）には介入しません。当事者同士での解決を基本とします。
* **規約違反への対応**  
    前項にかかわらず、第3条に定める「禁止事項（窃盗・破壊・暴言等）」に該当する客観的な事実（ログ等）が確認された場合は、運営の判断により適切な処罰を行います。
* **両成敗**  
    トラブルが激化し、サーバー全体の運営や秩序に支障が出ると運営が判断した場合、理由の如何を問わず、当事者双方を処罰の対象とする場合があります。

### 第6条（寄付と特典）

* ユーザーは、任意でサーバーへの寄付を行うことができます。
* 寄付の合計金額に応じて「VIP」「GOD」等の特典（ロール）が付与されます。**MinecraftのEULA（利用規約）を遵守するため、これらの特典はゲームバランスを著しく崩さない範囲（装飾や利便性向上など）に限定されます。**
* いかなる理由があっても、一度支払われた寄付金の返金は行いません。また、規約違反によりBANされた場合でも返金はなされません。

### 第7条（違反時の措置）

**本規約に違反した場合、運営は以下の措置を講じることができます。**

* 警告: 改善を求める注意喚起。
* 一時的な利用制限: Kick、タイムアウト、一時的なサーバー接続禁止。
* 永久利用停止（BAN）: 本サーバーおよび関連コミュニティへの永久的なアクセス禁止。
* BAN等の重い処罰は、STAFFの独断ではなくOWNERの決裁により執行されます。
* 処罰の際、適用された規約条項については提示しますが、詳細な調査手法やログ等の証拠については、再発防止およびセキュリティ維持の観点から非公開とする場合があります。

### 第8条（免責事項）

* 運営は、本サーバーの利用により生じたユーザーの損害（データ消失、精神的苦痛等）について、一切の責任を負いません。**運営は復旧の義務は負いませんが、運営の判断により善意でロールバック（復旧）を行う場合があります。**
* 本規約は予告なく変更される場合があります。変更後の規約は、本サーバーまたは公式サイト上に掲載された時点で効力を生じるものとします。
