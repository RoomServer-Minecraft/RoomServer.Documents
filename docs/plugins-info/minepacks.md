
# Minepacks

## Minepacksとは？

「Minepacks」とは、バックパックを追加するプラグインです。  
これにより、持ち歩けるアイテムの量を増やすことができます。  
さらに、バックパックに入れたアイテムはロストしません。

## 使い方

* インベントリのバックパック上で右クリック
* 手にバックパックを持って右クリック
