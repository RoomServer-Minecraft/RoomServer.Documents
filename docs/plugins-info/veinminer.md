# VeinMiner

## VeinMinerとは？

「VeinMiner」とは、一括破壊ができるようになるプラグインです。  
特定の種類のブロックを採掘すると、繋がっているブロックを壊すことができます！  
Veinminerは様々なMODパックやUHCのようなサバイバルpvpゲームモードでよく使われる機能で、快適なプレイをすることができます。
（鉱石や原木、葉ブロックや一部の作物などに対応しています。）  

## 使い方

1. しゃがむ
2. 掘りたい対象のブロックを掘る
3. 破壊完了！
