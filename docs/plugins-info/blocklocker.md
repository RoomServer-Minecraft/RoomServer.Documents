# BlockLocker

## BlockLockerとは？

「BlockLocker」とは、特定のブロックを他のプレイヤーから保護するプラグインです。
（チェストやドア、ホッパーや金床など、ブロックエンティティに対応しています。）

## 使い方

1. しゃがまずに保護したいブロックに向けて看板を置く
2. 保護完了！

## タグ一覧

* `[More Users]`を入力し、他の行にプレイヤー名を書くことで、アクセス可能なプレイヤーを追加することができます。
* `[Redstone]`で、レッドストーンによる干渉が可能になります。（ホッパーなど）
* `[Timer:1]`で、1秒後に自動で閉まるドアを作成できます。（秒数は変更可能）

## 対応している機能

* **複数ブロックに対応**: 片側だけサインを貼れば、繋がっているドアやラージチェストをまとめて保護できます&#x20;
* **UUIDに対応**: プレイヤー名の変更後も保護は解除されず、サインの裏側にUUID情報が隠し保存されます
* **グループに対応**: Permissions、スコアボードチームや、その他とのプラグインを連携しながらグループ単位のアクセス制御が可能
* **WorldEdit対応**: WorldEdit等で移動しても、保護が維持されます。
