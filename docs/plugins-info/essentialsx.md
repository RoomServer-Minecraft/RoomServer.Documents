# EssentialsX

## EssentialsXとは？

ホーム設定や他のプレイヤーへのテレポートリクエストなど、便利な機能が詰まったプラグインです。  
機能が多く全て書ききれていないので、公式ドキュメントを見ることをお勧めします。

## 特徴

* サーバー運営の必須機能を多数追加する多機能プラグイン。
* テレポート、ホーム設定、プラグインの英語の部分を日本語に翻訳するなど幅広く対応。
* 初心者からベテランまで使える便利コマンドが豊富。

## 導入目的

* サバイバル生活の利便性・効率性向上
* プレイヤー間の移動やコミュニケーションを円滑にする
