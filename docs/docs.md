# 📚 RoomServer (RSM) 公式ドキュメント・ルールブック

ようこそRoomServerへ！  
当サーバーは、プレイヤーの皆さんが快適に、かつ平等にプレイを楽しめる環境を作ることを目的としています。  
このドキュメントでは、サーバーの基本情報、ルール、システムについてまとめています。

## 🏛️ 第1章：サーバー基本情報と理念

### 1. サーバー情報  

**正式名称**: RoomServer（略称: RSM）

**対応エディション**: Java版 および 統合版（クロスプレイ対応）

**参加条件**: 誰でも参加可能なパブリックサーバー（ホワイトリスト不要）

### 2. コミュニティの理念と対象層  

* **コアバリュー**  
    「自由な空気」と「お互いへのリスペクト」。  
    過度なルールで縛るのではなく、プレイヤーそれぞれの遊び方を尊重します。  
    ただし、自由であっても「他者を不快にさせない」「他者の作品を尊重する」という思いやりの精神を大前提とします。

* **メインターゲット**  
    中学生・高校生を中心とした活気のあるプレイヤー層。  
    初心者も大歓迎ですが、実態としては中〜上級者が多く、高度な建築や技術が飛び交う環境です。

* **プレイスタイルの推奨**  
    ボイスチャット等を用いたグループ活動は許可されていますが、参加型サーバーの性質上、  
    **「サーバー全体での交流（ソロプレイヤーも含めて一緒にプレイするスタイル）」** を強く推奨しています。

### 3. ゲームモード構成

Lobby（ロビー）サーバーから、遊びたいモードのNPC（プレイヤーの見た目をしたキャラクター）を右クリックするだけで瞬時に移動できます。

* **素のサバイバル**  
    バニラに近い、マインクラフト本来の魅力を楽しむ基本モード。

* **なんでもありサバイバル**  
    PvP、戦争、略奪、自由な建築など、型に縛られない自由な旅が可能な特区。

* **経済（シンプル）サバイバル**  
    公式ショップ、職業（Job）、スキルシステム等を導入し、やり込み要素を楽しめるモード。**※順次リリース**

* **ミニゲーム**  
    サバイバルの息抜きや交流のためのコンテンツ。**※順次リリース**

## ⚖️ 第2章：ゲーム内ルールと利用規約

快適なプレイ環境維持のため、以下のルールを定めています。  
違反が確認された場合、運営の裁量により処罰（警告・一時BAN・永久BAN）の対象となります。

### 🚫 絶対禁止事項

* **チート・不正ツールの使用**  
    X-Ray、マクロ機能、不正クライアント等の使用（※通常のプレイを補助する軽量化MOD等は許可）。

* **ネザー岩盤上への立ち入り**  
    統合版プレイヤーがアクセスできず不公平になるため、原則禁止。

* **VPNの無断使用**  
    海外からの接続等でやむを得ない場合は、事前にお問い合わせにて許可を得ること。

* **荒らし・不適切な発言**  
    暴言、差別的発言、わいせつな表現、意図的な進行妨害。

* **運営への妨害**  
    サーバーシステムへの不正アクセス、DDoS攻撃、運営の指示に従わない行為。

* **個人情報の開示**  
    自身または他者の個人情報を公開する行為。

### ⚙️ サーバー負荷・システムに関するルール

* **装置・トラップ**  
    作成に特段の制限は設けませんが、サーバーに過度な負荷がかかり運営が困難と判断された場合は、一時的な制限や施設の撤去・停止を求める場合があります。

* **サブアカウントの利用**  
    利用は許可されていますが、専用申請フォームからの事前報告が必須です。無申告での複数アカウント接続は規約違反となります。

### 🎥 配信・動画投稿について

YouTubeやTwitch等での配信・動画投稿は許可されています。  
ただし、他プレイヤーの映り込み等に関する取り扱いは、他プレイヤーへの配慮に責任を持つものとし、本サーバーは責任を負いません。

### ⚠️ トラブル対応と免責事項（重要）

* **不介入の原則**  
    プレイヤー間のトラブル（アイテムの窃盗、揉め事など）に対して、運営は原則として介入しません。   
    当事者同士での解決をお願いします。

* **両者処罰の可能性**  
    トラブルが過熱し、サーバー運営に支障が出ると判断された場合、理由を問わず当事者双方をBAN等の処罰対象とする可能性が高くなります。

* **免責**  
    運営は、本サーバーの利用により生じた一切の損害について責任を負いません。

## 🛡️ 第3章：サーバー内システム（保護・復旧）

### 1. 財産・建築物の保護（荒らし対策）

自分の家やアイテムを守るため、以下のシステムを活用してください。

* **チェストの保護（セルフ）**  
    保護したいチェスト等に向かって、「看板」を持って右クリックするだけでロックが可能です。

* **建築物・土地の保護（申請制）**  
    プレイヤー自身でコマンドを打つことはできません。  
    保護したい土地や建物がある場合は、Discord内の「チケット」チャンネルから運営へ申請してください。

### 2. 被害発生時のログ調査と復旧（ロールバック）

サーバーにはブロックの設置・破壊・チェスト操作のログを記録するツールが導入されています。   
「あからさまな悪意のある荒らし・無差別な破壊や窃盗」の被害に遭った場合は、チケットからお問い合わせください。

スタッフが現場のログを調査し、被害の度合いを確認した上で、  
該当箇所を以前の状態に戻す「**ロールバック（復旧）**」を行う場合があります（ユーザー側の過失状況等も考慮して判断します）。

## 👑 第5章：Discordと運営（STAFF）体制

### 1. Discordチャンネルについて

Discordはゲーム内とのシームレスなコミュニケーションの場です。

* **Minecraft連携チャンネル**  
    ゲーム内のチャットと連動しており、ログインしていなくても中の様子がわかります。

* **Discord固有のルール**  
    暴言・スパムの禁止に加え、プレイヤー間トラブルのDiscordへの持ち込み（公開晒し等）は控えてください。

### 2. STAFF（運営陣）について

RoomServerの「STAFF」は、専用の試験を突破し、厳格な条件（法律・規約の遵守、報連相の徹底など）をクリアした信頼できるモデレーター陣です。  
以下の部門に分かれてサポートを行っています。

**チェッカー**: マイクラ内での新規案内、荒らしへの即時対応。

**ヘルパー**: お問い合わせ（チケット）対応、総括管理部の補佐。

**広報部**: SNSマーケティング、配信、動画編集など。

### 3. モデレーション権限

**STAFFの権限**: 規約違反を確認した場合、「キック、タイムアウト、一時的な隔離」などの一時的な処罰を独断で実行できます。

**OWNER決裁**: アカウントの永久停止（BAN）などの重い処罰は、STAFFの独断では行わず、必ずOWNER（総括管理部）への相談と決裁を経て実行されます。
